# 01 - ภาพรวมระบบ

## 🎯 ปัญหาที่แก้

| ปัญหา | วิธีแก้ |
|-------|--------|
| เอกสารกระจัดกระจาย | รวมทุกอย่างใน "กล่อง" เดียว |
| ไม่รู้ว่าขาดเอกสารอะไร | ระบบติดตามอัตโนมัติ |
| หา WHT ไม่เจอ | WHT Tracking |
| ส่งบัญชียุ่งยาก | Export Excel/CSV พร้อมไฟล์ |
| กรอกข้อมูลซ้ำ | AI อ่านเอกสารให้ |

---

## 📦 Core Concept: กล่องเอกสาร

```
📦 "กล่อง" = 1 ธุรกรรม (เช่น จ่ายค่าบริการ IT)

กล่องเก็บ:
├── 💳 สลิปโอนเงิน
├── 📋 ใบกำกับภาษี
├── 📝 หนังสือหัก ณ ที่จ่าย (WHT)
└── 📄 เอกสารอื่นๆ ที่เกี่ยวข้อง

ระบบจะบอกว่า: "กล่องนี้ยังขาด ใบกำกับภาษี"
```

---

## 🔄 Flow หลัก

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   1. 📤 อัปเอกสาร                                               │
│      └── อัปอะไรก็ได้ (สลิป, ใบกำกับ, บิลเงินสด, อื่นๆ)        │
│                                                                 │
│   2. 🤖 AI อ่านและจัดการ                                        │
│      ├── แยกประเภท: สลิป? ใบกำกับ? บิลเงินสด?                  │
│      ├── ดึงข้อมูล: ยอด, วันที่, ร้านค้า                       │
│      └── สร้างกล่องใหม่ หรือ เพิ่มในกล่องเดิม                  │
│                                                                 │
│   3. 👁️ User ตรวจสอบ (ถ้าต้องการ)                               │
│      └── ยืนยัน หรือ แก้ไข                                     │
│                                                                 │
│   4. 📊 ระบบติดตาม                                              │
│      └── "ยังขาด: ใบกำกับ" / "รอส่ง WHT"                       │
│                                                                 │
│   5. 📤 Export (เมื่อพร้อม)                                     │
│      └── Excel/CSV + ไฟล์เอกสาร                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 👥 Roles

| Role | ทำอะไรได้ |
|------|-----------|
| **staff** | อัปเอกสาร, ดูกล่องตัวเอง |
| **accounting** | ตรวจกล่อง, Export, ดูทุกกล่อง |
| **admin** | ทุกอย่าง + ตั้งค่าระบบ |
| **owner** | ทุกอย่าง + Billing |

---

## ✨ จุดขาย

1. **อัปง่าย** - ถ่ายรูป/เลือกไฟล์ AI จัดการให้
2. **ไม่ตกหล่น** - รู้ว่าขาดเอกสารอะไร
3. **WHT Tracking** - ติดตาม WHT ส่ง/รับ
4. **Export ง่าย** - Excel + ไฟล์พร้อมส่งบัญชี
5. **Multi-org** - รองรับหลายบริษัท

---

## ❌ สิ่งที่ระบบไม่ทำ

| ไม่ทำ | เหตุผล |
|-------|--------|
| คำนวณ VAT | บัญชีทำในโปรแกรมบัญชี |
| คำนวณ % WHT | บัญชีเลือกเอง |
| บันทึกบัญชี | ใช้โปรแกรมบัญชีจริง (PEAK, Express) |
| ออกใบกำกับ | ไม่ใช่ scope |

**ระบบนี้คือ**: Document Hub + Tracking (ไม่ใช่ Accounting Software)
