## 5.1 หน้า Upload — สร้างกล่องเอกสาร

### ฟีเจอร์หลัก
- **📦 Record First**: สร้างกล่องก่อน → ใส่เอกสารทีหลัง
- **📄 Multi-file Support**: อัปโหลดหลายไฟล์ต่อ 1 กล่อง
- **💾 Draft Mode**: บันทึกไว้ก่อน กลับมาเพิ่มทีหลังได้
- **📷 Camera Capture**: ถ่ายรูปจากกล้องโดยตรง
- **🎤 Voice Note**: บันทึกเสียงเป็นหมายเหตุ

### Upload Flow (กล่องเอกสาร)

```
┌─────────────────────────────────────────────────────────────────┐
│                   📦 สร้างกล่องเอกสาร                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Step 1             Step 2             Step 3                   │
│  ┌───────┐         ┌───────┐         ┌───────┐                 │
│  │📦 สร้าง│ ─────▶  │📄 ใส่  │ ─────▶  │✅ ส่ง │                 │
│  │ กล่อง │         │ เอกสาร│         │      │                 │
│  └───────┘         └───┬───┘         └───────┘                 │
│     ●──────────────────●──────────────────●                     │
│                        │                                        │
│                        ▼                                        │
│                  [บันทึก Draft]                                 │
│              กลับมาเพิ่มทีหลังได้                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Step Details

**Step 1: สร้างกล่อง**
- ประเภทเอกสาร (Radio buttons): ใบเสร็จ / ใบกำกับ / ใบแจ้งหนี้ / อื่นๆ
- หมวดค่าใช้จ่าย (Dropdown)
- ศูนย์ต้นทุน/แผนก (Dropdown - optional)
- ร้าน/คู่ค้า (Text + autocomplete)
- **มีหัก ณ ที่จ่ายหรือไม่?** (Toggle - ระบบแนะนำอัตโนมัติตามหมวด)
- 👉 กด "สร้างกล่อง" → ได้ Document (status: draft)

**Step 2: ใส่เอกสารลงกล่อง**
- ปุ่มถ่ายรูป (Camera)
- ปุ่มเลือกไฟล์ (Gallery/Files)
- **อัปโหลดได้หลายไฟล์** (เช่น หน้า 1, 2, 3 หรือ หน้า-หลัง)
- ลากเรียงลำดับไฟล์ได้
- ตั้งไฟล์หลัก (Primary) สำหรับ thumbnail
- ✅ กด "บันทึก Draft" → กลับมาเพิ่มทีหลังได้

**Step 3: กรอกยอดและส่ง**
- ยอดเงิน (Number input)
- VAT (Toggle: 7% / ไม่มี)
- วันที่เอกสาร (Date picker)
- วิธีชำระ (Radio buttons)
- หมายเหตุ (Optional textarea)
- กด "📤 ส่งให้บัญชี"

## 5.2 Inbox / To Review (บัญชี)

### Layout

```
┌─────────────────────────────────────────────────────────────────┐
│  ┌──── Transaction Type ────┐                                   │
│  │ ● ทั้งหมด │ รายจ่าย │ รายรับ │                               │
│  └───────────────────────────┘                                  │
├─────────────────────────────────────────────────────────────────┤
│  🔍 Search    [Filter ▼]    [Export ▼]    [Bulk Actions ▼]      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ☐ │ 📄 DOC-202601-0001 │ 7-Eleven │ ฿150 │ 🔵 Pending    │   │
│  │   │ สลิป │ ค่าเดินทาง │ สมชาย │ 2 ชม. ก่อน │ ⚠️ Duplicate │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ☐ │ 📄 DOC-202601-0002 │ Grab │ ฿350 │ 🟠 Need Info     │   │
│  │   │ ใบเสร็จ │ ค่าเดินทาง │ สมหญิง │ 1 วัน ก่อน │ 🔥 Due Soon│   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ ☐ │ 📄 DOC-202601-0003 │ Apple │ ฿45,000 │ 🟢 Ready      │   │
│  │   │ ใบกำกับ │ อุปกรณ์ │ CEO │ 3 วัน ก่อน                    │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Filters
- **ประเภทธุรกรรม**: รายจ่าย (Expense) / รายรับ (Income) / ทั้งหมด ⭐
- Status: pending_review / need_info / ready_to_export
- ประเภทเอกสาร
- หมวด
- ผู้ส่ง
- ศูนย์ต้นทุน
- ช่วงวันที่
- ช่วงยอดเงิน
- **สถานะชำระ** (Income): รอชำระ / รับแล้ว / เกินกำหนด ⭐

### Badges
- **⚠️ Duplicate Warning**: แสดงเมื่อตรวจพบอาจซ้ำ
- **🔥 Due Soon**: แสดงเมื่อใกล้ deadline

### Bulk Actions
- ✅ Bulk Approve → เปลี่ยนสถานะเป็น ready_to_export
- ❌ Bulk Reject → เปลี่ยนสถานะเป็น rejected + ใส่เหตุผล
- 📤 Bulk Export → Export เอกสารที่เลือกเป็น Excel/ZIP
- 🏷️ Bulk Assign → กำหนดหมวด/ศูนย์ต้นทุนพร้อมกัน
- 📁 Bulk Add to Group → เพิ่มเข้า Expense Group
