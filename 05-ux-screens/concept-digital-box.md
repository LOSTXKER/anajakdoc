# 📦 Core Concept: กล่องเอกสารดิจิทัล (Digital Document Box)

> **จุดขายหลักของระบบ**: "อัปเอกสาร → AI จัดให้ → ติดตามจนครบ"

---

## 🎯 แนวคิดหลัก

ระบบเป็น **Document Hub + Tracking** ที่ใช้ **AI-first** approach

```
┌─────────────────────────────────────────────────────────────────┐
│                    📦 กล่องเอกสารดิจิทัล                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   📄 "อัปก่อน" = อัปเอกสารอะไรก็ได้ (สลิป/ใบกำกับ/อื่นๆ)         │
│   🤖 "AI จัด" = AI อ่านและจัดประเภท/สร้างกล่องให้              │
│   📦 "กล่อง" = ธุรกรรม 1 รายการ (รวมเอกสารที่เกี่ยวข้อง)        │
│   📊 "ติดตาม" = รู้ว่ายังขาดเอกสารอะไร ไม่มีอะไรตกหล่น         │
│                                                                 │
│   ✅ Value: "ไม่มีเอกสารตกหล่น + ติดตามได้จนครบ"               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 🔄 Flow หลัก: AI-First

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   1. 📤 อัปเอกสาร                                               │
│      └── อัปอะไรก็ได้ (สลิป, ใบกำกับ, ใบเสร็จ...)              │
│                                                                 │
│   2. 🤖 AI อ่านและจัดการ                                        │
│      ├── แยกประเภท: สลิป? ใบกำกับ? ใบแจ้งหนี้?                 │
│      ├── ดึงข้อมูล: ยอด, วันที่, ร้านค้า                       │
│      └── สร้างกล่อง หรือ Match กับกล่องเดิม                    │
│                                                                 │
│   3. 👁️ User ตรวจสอบ                                            │
│      └── ยืนยัน หรือ แก้ไขถ้าจำเป็น                            │
│                                                                 │
│   4. 📊 ระบบติดตาม                                              │
│      └── แจ้งว่า "ยังขาดเอกสารอะไร"                            │
│                                                                 │
│   5. 📤 ส่งบัญชี (เมื่อพร้อม)                                   │
│      └── Export ไปลงโปรแกรมบัญชี                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 📄 เอกสารแต่ละประเภท → รู้อะไร?

### สลิปโอนเงิน (Payment Slip)
```
┌─────────────────────────────────────────────────────┐
│ 🧾 สลิปโอนเงิน                                      │
├─────────────────────────────────────────────────────┤
│ รู้:     ✅ ยอดโอน, วันที่, โอนให้ใคร              │
│ ไม่รู้:  ❓ มี VAT ไหม?                             │
│          ❓ ต้องหัก ณ ที่จ่ายไหม?                   │
│          ❓ เป็นค่าอะไร (สินค้า/บริการ)?           │
├─────────────────────────────────────────────────────┤
│ สถานะ:  ⏳ รอใบกำกับภาษี                           │
└─────────────────────────────────────────────────────┘
```

### ใบกำกับภาษี (Tax Invoice)
```
┌─────────────────────────────────────────────────────┐
│ 📋 ใบกำกับภาษี                                      │
├─────────────────────────────────────────────────────┤
│ รู้:     ✅ ยอด, VAT, วันที่, ร้านค้า, Tax ID      │
│          ✅ เป็นค่าอะไร                             │
│          ✅ เครม VAT ได้                            │
├─────────────────────────────────────────────────────┤
│ สถานะ:  ✅ ข้อมูลครบ                               │
└─────────────────────────────────────────────────────┘
```

### หนังสือหัก ณ ที่จ่าย (WHT Certificate)
```
┌─────────────────────────────────────────────────────┐
│ 📝 หนังสือหัก ณ ที่จ่าย                             │
├─────────────────────────────────────────────────────┤
│ รู้:     ✅ ยอดหัก, อัตรา, ผู้รับ                   │
├─────────────────────────────────────────────────────┤
│ ติดตาม: ⏳ ส่งให้ผู้รับแล้วยัง?                     │
│         ⏳ ผู้รับยืนยันได้รับยัง?                   │
└─────────────────────────────────────────────────────┘
```

## 📊 ตัวอย่าง: Timeline ชีวิตจริง

```
วันที่ 1: Staff จ่ายเงินร้าน A
         └── อัปสลิป → AI สร้างกล่อง
         └── สถานะ: "จ่ายเงินแล้ว ฿10,700 - รอใบกำกับ"

วันที่ 5: ได้ใบกำกับจากร้าน A
         └── อัปใบกำกับ → AI match กับกล่องเดิม
         └── รู้: VAT ฿700, เป็นค่าบริการ IT
         └── สถานะ: "มี VAT - รอส่ง WHT"

วันที่ 7: ออก WHT และส่งให้ร้าน A
         └── อัป WHT cert + กดส่ง
         └── สถานะ: "WHT ส่งแล้ว - รอยืนยัน"

วันที่ 10: ร้าน A ยืนยันได้รับ WHT
         └── กดยืนยัน
         └── สถานะ: "✅ เอกสารครบ 100%"
```

## 🤖 AI ทำอะไรบ้าง?

### 1. อ่านเอกสาร
| ข้อมูลที่ AI อ่าน | ตัวอย่าง |
|------------------|----------|
| ประเภทเอกสาร | สลิป / ใบกำกับ / ใบเสร็จ |
| ยอดเงิน | ฿10,700 |
| วันที่ | 15/01/2569 |
| ร้านค้า/ผู้รับเงิน | บริษัท ABC จำกัด |
| Tax ID | 0105556000XXX |
| รายละเอียด | ค่าบริการ IT เดือน ม.ค. |

### 2. สร้างกล่องอัตโนมัติ
- อัปสลิปใหม่ → สร้างกล่องใหม่
- รู้ว่าโอนให้ใคร, ยอดเท่าไหร่, วันไหน

### 3. Match เอกสารที่มาทีหลัง
- อัปใบกำกับ → หาว่า match กับกล่องไหน
- ตาม: ร้านเดียวกัน + ยอดใกล้เคียง + ช่วงเวลาใกล้กัน

### 4. แนะนำสิ่งที่ต้องทำ
- "ธุรกรรมนี้น่าจะมี WHT - ต้องการออกหนังสือไหม?"
- "ยังไม่มีใบกำกับภาษี - รอรับจากร้าน"

## 🔄 สถานะกล่อง

```
📝 กำลังรวบรวม  →  📤 ส่งบัญชีแล้ว  →  ✅ Export แล้ว
```

| สถานะ | ความหมาย |
|-------|----------|
| 📝 กำลังรวบรวม | เอกสารยังไม่ครบ / กำลังรอเอกสารเพิ่ม |
| 📤 ส่งบัญชีแล้ว | เอกสารครบ ส่งให้บัญชีตรวจ |
| ✅ Export แล้ว | บัญชี export ไปลงโปรแกรมบัญชีแล้ว |

| **Export** | ส่งข้อมูลไปโปรแกรมบัญชี |
