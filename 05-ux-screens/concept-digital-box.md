# 📦 Core Concept: กล่องเอกสารดิจิทัล (Digital Document Box)

> **จุดขายหลักของระบบ**: "สร้างกล่อง → ใส่เอกสาร → ส่งให้บัญชี"

---

## 🎯 แนวคิดหลัก

ระบบใช้แนวคิด **กล่องเอกสารดิจิทัล** ที่ทำให้การจัดการเอกสารง่ายเหมือนใส่ของลงกล่อง

```
┌─────────────────────────────────────────────────────────────────┐
│                    📦 กล่องเอกสารดิจิทัล                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   คิดแบบง่ายๆ:                                                  │
│                                                                 │
│   📦 "กล่อง" = รายการค่าใช้จ่าย 1 รายการ                        │
│   📄 "เอกสาร" = ไฟล์รูป/PDF ที่ใส่ในกล่อง (กี่ไฟล์ก็ได้)         │
│   🏷️ "ป้าย" = ข้อมูลที่ติดบนกล่อง (หมวด, ยอด, วันที่)            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```


---

## ✨ ข้อดีของแนวคิดนี้

| ข้อดี | คำอธิบาย |
|-------|----------|
| **ใส่หลายไฟล์ได้** | ใบเสร็จหลายหน้า, หน้า-หลัง, เอกสารเกี่ยวข้อง |
| **เก็บไว้ก่อนได้** | Draft Mode - กลับมาเพิ่มทีหลัง |
| **เข้าใจง่าย** | คนทั่วไปเข้าใจว่า "ใส่ของลงกล่อง" |
| **บัญชีจัดการง่าย** | เห็นทุกอย่างในที่เดียว |

## 🎨 Branding Keywords

- **"กล่องเอกสารดิจิทัล"** - ใช้สื่อสารกับผู้ใช้ทั่วไป
- **"Digital Document Box"** - ใช้สื่อสารกับลูกค้าองค์กร
- **"สร้างกล่อง → ใส่เอกสาร → ส่งให้บัญชี"** - Tagline หลัก
- **"ใส่กี่ไฟล์ก็ได้ เก็บไว้ก่อนก็ได้"** - จุดขายย่อย
